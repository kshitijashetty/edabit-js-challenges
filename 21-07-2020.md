[Summing a Slice](https://edabit.com/challenge/B3FR3P7g8NyTg7t8b)
```javascript
function sliceSum(arr, n) {
	return n==0?0:arr.slice(0,n).reduce((s,x)=>s+=x);
}
```

function sumOfCubes(nums) {

}
